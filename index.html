Upload Music

To add songs to the library:

Method 1: GitHub Upload (Recommended)

  1. Go to the TWS Music Storage repository on GitHub
  2. Click "Add file" ‚Üí "Upload files"
  3. Drag & drop your MP3 file

  5. Click "Commit changes"

Method 2: Request Upload

If you don't have GitHub access, contact the admin with:

  • Your MP3 file
  • Desired song name
  • Your display name (optional)

Naming Rules:

  • Use only letters, numbers, and dashes
  • No spaces (use dashes instead)
  • Example: my-cool-song.mp3
  • Max file size: 100MB

SA:MP Streaming Guide

Step-by-Step Tutorial:

  1. Find a song in the Songs tab
  2. Copy the URL by clicking "Copy URL" button
  3. In SA:MP, open chat and type:
    /stream PASTE_YOUR_URL_HERE
  4. Press Enter - the music will start playing!

Troubleshooting:

If music doesn't play:

  • Make sure URL starts with https://
  • URL must end with .mp3
  • Some servers may have streaming disabled
  • Try a different song to test

Example SA:MP Command:

/stream https://raw.githubusercontent.com/binosko/tws-music-storage/main/example-song.mp3
